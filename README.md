# Rohit Raju
 
**AI/ML Engineer building production LLM and agentic systems.**
 
I work on the unglamorous parts of making agents reliable: tool-calling architectures, RAG, NL-to-SQL with real guardrails, and knowing when a task belongs in deterministic code instead of model reasoning. MS in Data Science from CU Boulder, 5 peer-reviewed NLP/ML publications, and a production multi-agent system deployed across 500+ retail stores.
 
Currently building LLM discourse-analysis pipelines for K-12 classrooms at CU Boulder's Institute of Cognitive Science.
 
🌐 [rohitraju.com](https://rohitraju.com) &nbsp;·&nbsp; 📍 Austin, TX

### Featured work
 
**[Sherlogs](https://github.com/rohitraju1966/Sherlogs)** — Sherlock for logs. An agent that reads raw logs and pinpoints which microservice caused an incident. 92.2% top-1 accuracy on the RCAEval RE3 benchmark (vs. 83.3% baseline). Built with Drain3, DuckDB, and LangGraph.
 
**[Census Query Agent](https://github.com/rohitraju1966/Census-Query-Agent)** — Chat-based agent that answers US population questions over the ACS dataset. Agentic NL-to-SQL on Snowflake Cortex with a self-correcting query loop and multi-layer safety guardrails.
